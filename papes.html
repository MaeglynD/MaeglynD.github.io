Papes.io

Papes.io was made to test out some of the features of VueJS, Vuetify, Vue router (nuxt), webpack, babel and so on. The site is a wallpaper sharing / showcasing site, equipped with a ‘trending’ page and category section where wallpapers could be separated through tags and sorted accordingly. At first glance, the site kinda looks like youtube’s layout a little right? Hopefully so, I used a framework called vuetify which is based solely on material design rules (Google's own design schema). Vuetify is a Vue framework which, unlike css frameworks, can manipulate the DOM through attributes and components. The developers really go above and beyond with some of the small details in the framework, for example - try playing around with the ‘create account’ dialog, notice how the tab underline moves based on the tab currently active.

Try clicking on another page from the navigation drawer, watch the URL and the refresh status. That's right server side rendering baby, thanks to the amazing dev’s over at Nuxt you’ll never have to refresh papes again. Vue also allows us to render the navigation components once, with everything else on the page being part of a Vue view page or component. Nuxt really saves the day when it comes to live production implementation, its as easy as changing a few scripts in package.json, npm run generate and wallah. If you wanna check out the raw vue / vuetify code then check out my personal github, or if you wanna check out the live site’s code from github pages click here!
